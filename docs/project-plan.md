# Project Plan

## Goal

시각장애인이 온라인 서점에서 책을 찾고 구매 흐름을 이해할 수 있도록 돕는 앱을 설계합니다.

## Target Users

- 화면의 작은 글씨를 읽기 어려운 사용자
- 검색 결과와 구매 버튼 위치를 찾기 어려운 사용자
- 구매 단계에서 현재 진행 위치를 음성으로 확인하고 싶은 사용자

## Main Flow

1. 사용자가 책 제목 또는 저자를 입력합니다.
2. 앱이 검색 결과를 카드 형태로 정리합니다.
3. 사용자는 큰 버튼으로 책 상세 정보를 확인합니다.
4. 앱은 구매 단계와 주의 사항을 짧은 문장으로 안내합니다.
5. 사용자는 즐겨찾기에 저장하거나 구매 페이지로 이동합니다.
